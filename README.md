# Iron ore drying system

Simulação de um sistema de secagem de minério de ferro

# Stacks

- MPLABX = 6.00
- XC8 = 2.46
- PICSimLab = 0.9.2_241005

# Requisitos técnicos

- Resistências de 220V 1kW
- Recebe energia elétrica em 220V
- Sistema instalado em uma sala climatizada com 20 graus Celsius
- Sistema deve ter modo de operação automático e manual
- No modo manual é utilizado botões para ligar e desligar
- No modo automático a estufa defe ligar durante uma hora (10 segundos na simulação)
- Os modos e o status de funcionamento devem ser informados em um LCD
  
# Comando inteiro

Com base nos exemplos disponibilizados em aula, desenvolva um programa para
simular um sistema de secagem de minério de ferro. Requisitos: resistências de 220 V
e 1 kW, há disponibilidade de energia elétrica (220 V) no local e o sistema será instalado
em uma sala climatizada (temperatura controlada a 20 °C). O sistema deve ter dois
modos de operação: manual e automático. No modo manual o usuário pode ligar ou
desligar o sistema por meio de botões. No modo automático, a estufa deve ligar durante
uma hora (10 segundos efeito de simulação). O modo (“Manual” ou “Automático”) e o
status do sistema (“On” ou “Off”) deve ser informado em um LCD.
